Software Engineer

ເປັນອາຊີບໃນສາຍງານວິສະວະກຳທີ່ກ່ຽວຂ້ອງກັບການອອກແບບ, ການພັດທະນາ ແລະ ການບໍາລຸງຮັກສາລະບົບຊອບແວ ແລະ ແອັບພລິເຄຊັນ

    ບົດບາດໃນອົງກອນ:

    ສຳຄັນໃນຂະບວນການສ້າງ ແລະ ພັດທະນາຊອບແວ ທີ່ນຳສະເໜີຄວາມສາມາດ ແລະ ຟັງຊັນທີ່ຫຼາດຫຼາຍ ໃນດ້ານຕ່າງໆ ເຊັ່ນ: ແອັບພລິເຄຊັນມືຖື, ໂປຣແກຣມຄອມພິວເຕີ, ລະບົບຖານຂໍ້ມູນ ແລະ ອື່ນໆ ເຊິ່ງມີບົດບາດສຳຄັນໃນການສະໜັບສະໜູນການເຮັດວຽກຂອງອົງກອນ

    ທັກສະທີ່ຈຳເປັນ:

  • ໂຄງສ້າງຂໍ້ມູນ ແລະ ອັລກໍຣິທຶມ (Data Structures ແລະ Algorithms)
  • ການຂຽນໂຄ້ດ (Coding)
  • ສະຖາປັດຕະຍະກຳຊອບແວ (Software Architecture)
  • ຂະບວນທັດການຂຽນໂປຣແກຣມ (Programming Paradigms)
  • ການທົດສອບ (Testing)
  • ຖານຂໍ້ມູນ (Database)
  • ຄວາມປອດໄພຂອງເວັບ (Web Security)

Software Developer

ແມ່ນບຸກຄົນທີ່ມີຄວາມຮູ້ ແລະ ຄວາມສາມາດທີ່ຈຳເປັນຕໍ່ການສ້າງຊອບແວ ເຊິ່ງແມ່ນໂປຣແກຣມທີ່ເຮັດວຽກໃນ ເຄື່ອງຄອມພິວເຕີ, ສະມາດໂຟນ ຫຼື ອຸປະກອນເອເລັກໂຕຣນິກອື່ນໆ ໂດຍໄດ້ທຳການອອກແບບ, ວາງແຜນ ແລະ ຂຽນໂຄ້ດ (Code) ໂດຍໃຊ້ພາສາໂປຣແກຣມ (Programming Language) ທີ່ເໝາະສົມກັບຊອບແວນັ້ນໆ

    ບົດບາດໃນອົງກອນ:

    ເປັນສາຍອາຊີບທີ່ຈໍາເປັນຕໍ່ອົງການທຸກປະເພດ ທີ່ມີຄວາມຕ້ອງການນໍາເຕັກໂນໂລຊີມາໃຊ້ໃນການດໍາເນີນທຸລະກິດ ຫຼື ຕ້ອງການເຜີຍແຜ່ຂໍ້ມູນ ສິນຄ້າ ແລະ ບໍລິການຂອງຕົນເອງຜ່ານເວັບໄຊ ຫຼື ແອັບພລິເຄຊັນ ຕັ້ງແຕ່ອົງກອນຂະໜາດນ້ອຍຈົນເຖິງຂະໜາດໃຫຍ່.

    ທັກສະທີ່ຈຳເປັນ:

  • ໂຄງສ້າງຂໍ້ມູນ ແລະ ອັລກໍຣິທຶມ (Data Structures ແລະ Algorithms)
  • ການຂຽນໂຄ້ດ (Coding)
  • ທັກສະລະບົບປະຕິບັດການ(Operating System)
  • ຖານຂໍ້ມູນ (Database)
  • ຄວາມປອດໄພຂອງເວັບ (Web Security)

Web Developer

ແມ່ນຜູ້ອອກແບບ ແລະ ພັດທະນາເວັບໄຊໃຫ້ສາມາດໃຊ້ງານໄດ້ຢ່າງມີປະສິດທິພາບ ແລະສາມາດຕອບສະໜອງຄວາມຕ້ອງການຂອງຜູ້ໃຊ້ງານໄດ້ຢ່າງລົງຕົວ.

    ບົດບາດໃນອົງກອນ:

    ເຮັດໃຫ້ເວັບໄຊມີຄວາມໜ້າເຊື່ອຖື, ລູກຄ້າສາມາດເຂົ້າໃຊ້ງານໄດ້ຢ່າງສະດວກ ແລະວ່ອງໄວ, ຮອງຮັບການເຮັດ SEO (Search Engine Optimization) ໄດ້ຢ່າງມີປະສິດທິພາບ ແລະມີລະບົບຫຼັງບ້ານທີ່ດີ.

    ທັກສະທີ່ຈຳເປັນ:

  • ຮູ້ພາສາໂປຣແກຣມ
  • ຮູ້ວິທີຈັດການ Source Code
  • ມີຄວາມຮູ້ດ້ານ SEO
  • ມີຄວາມຮູ້ກ່ຽວກັບການອອກແບບ UX/UI
  • ມີການອອກແບບໜ້າເວັບ ໃຫ້ປັບຂະໜາດໄດ້ (Responsive Design)
