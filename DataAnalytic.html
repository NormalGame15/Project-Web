Data Scientist

ແມ່ນຜູ້ຊ່ຽວຊານທີ່ມີໜ້າທີ່ຮວບຮວມ ແລະ ນຳຂໍ້ມູນຕ່າງໆທີ່ມີຢູ່ມາວິເຄາະໂດຍໃຊ້ເຕັກນິກຂັ້ນສູງ ເຊັ່ນ: ເຕັກນິກທາງສະຖິຕິ, ການເຮັດ Data Mining, ການໃຊ້ Algorithms ຫຼື ການສ້າງລະບົບ Machine Learning.

    ບົດບາດໃນອົງກອນ:

    ແມ່ນຜູ້ທີ່ສ້າງມູນຄ່າຈາກຂໍ້ມູນ ເພື່ອໃຫ້ອົງກອນສາມາດນຳຂໍ້ມູນໄປໃຊ້ໃນມຸມທີ່ຫຼາກຫຼາຍ ແລະ ມີປະສິດທິພາບຫຼາຍຂຶ້ນ ໂດຍນຳຂໍ້ມູນມາວິເຄາະໃນມຸມຕ່າງໆ, ນຳຂໍ້ມູນມາສ້າງເປັນໂມເດລ (Model) ເພື່ອໃຊ້ປະໂຫຍດໃນມຸມຕ່າງໆ ແຕ່ທັງນີ້ກໍຂື້ນຢູ່ກັບແຕ່ລະອົງກອນ.

    ທັກສະທີ່ຈຳເປັນ:

  • ຕ້ອງມີຄວາມຮູ້ດ້ານຄະນິດສາດ ແລະ ສະຖິຕິ ບໍ່ວ່າຈະເປັນ Algebra, Probability ແລະ Calculus
  • ຕ້ອງມີຄວາມຮູ້ດ້ານການໃຊ້ເຄື່ອງມືຕ່າງໆ ເຊັ່ນ: Data Mining ແລະ ການເຮັດ Machine Learning
  • ເຂົ້າໃຈໂປຣແກຣມຖານຂໍ້ມູນ ເຊັ່ນ: SQL Server, Oracle ແລະ SAP
  • ມີຄວາມສາມາດໃນການຂຽນພາສາໂປຣແກຣມ ບໍ່ວ່າຈະເປັນ SQL, Python, ພາສາ R, Json ຫຼື JavaScript
  • ສາມາດສ້າງ Dashboard ໂດຍການສ້າງ Power BI ຫຼື Tableau

Data Engineer

ແມ່ນຜູ້ທີ່ມີໜ້າທີ່ນໍາຂໍ້ມູນຈາກແຫຼ່ງຂໍ້ມູນ (Data Source) ໄປເກັບໄວ້ໃນລະບົບເກັບຂໍ້ມູນ ເຊັ່ນ Database, Data Warehouse ຫຼື Data Lake ເພື່ອໃຫ້ຄົນໃນອົງກອນນໍາໄປວິເຄາະຂໍ້ມູນໄດ້ຢ່າງງ່າຍດາຍ.

    ບົດບາດໃນອົງກອນ:

    ບົດບາດທີ່ສຳຄັນຂອງ Data Engineer ແມ່ນຜູ້ສ້າງລະບົບການເກັບຮວບຮວມ, ຈັດເກັບ ແລະ ວິເຄາະຊຸດຂໍ້ມູນທີ່ກ່ຽວຂ້ອງໃຫ້ແກ່ອົງກອນ; ພ້ອມກັນນັ້ນຍັງເປັນຜູ້ທີ່ນໍາຂໍ້ມູນທີ່ກ່ຽວຂ້ອງມາເປັນເຄື່ອງມືໃນການຊ່ວຍໃຫ້ທຸລະກິດຂອງອົງກອນຈະເລີນເຕີບໂຕ

    ທັກສະທີ່ຈຳເປັນ:

  • ທັກສະດ້ານຄະນິດສາດ ແລະ ສະຖິຕິ
  • ປະສົບການໂດຍກົງກັບການອອກແບບຖານຂໍ້ມູນ SQL
  • ເຂົ້າໃຈເລື່ອງສະຄຣິບ ແລະ ເວັບໄຊ ເຊັ່ນ: json, xml, JavaScript, HTML5
  • ເຂົ້າໃຈຂະບວນການ ETL ແລະ ການໃຊ້ເຄື່ອງມືເຊັ່ນ: SSIS, Power Query
  • ສາມາດໃຊ້ເຄື່ອງມືໃນການສ້າງລາຍງານ ແລະ ສ້າງ Dashboard ເຊັ່ນ: Power BI, Tableau, Qlik
  • ⁠ຄວາມຊໍານານດ້ານເຕັກນິກດ້ວຍໂມເດລຂໍ້ມູນ ແລະ ການຂຸດຄົ້ນຂໍ້ມູນ (Data Mining)
  • ຄວາມຮູ້ກ່ຽວກັບພາສາໂປຣແກຣມ (ເຊັ່ນ: Python, Java ຫຼື .NET)

Data Analyst

ເປັນຜູ້ຊ່ຽວຊານທີ່ມີໜ້າທີ່ໃນການເກັບຮວບຮວມ, ວິເຄາະ, ຕີຄວາມ ແລະ ນໍາສະເໜີຂໍ້ມູນ ໂດຍໃຊ້ທັກສະທາງຄະນິດສາດ, ສະຖິຕິ ແລະ ເຕັກໂນໂລຊີ ເພື່ອຊ່ວຍອົງກອນຕ່າງໆ ໃນການວິເຄາະຂໍ້ມູນຂະໜາດໃຫຍ່.

    ບົດບາດໃນອົງກອນ:

    ມີບົດບາດສຳຄັນໃນການວິເຄາະ ແລະ ນໍາຂໍ້ມູນມາໃຊ້ປະໂຫຍດ ເພື່ອພັດທະນາ ແລະ ແກ້ໄຂບັນຫາໃນອົງກອນ.

    ທັກສະທີ່ຈຳເປັນ:

  • ຄວາມຮູ້ດ້ານຄະນິດສາດ, ສະຖິຕິ ແລະ ການວິເຄາະຂໍ້ມູນ
  • ທັກສະດ້ານເຕັກໂນໂລຊີ ເຊັ່ນ ຖານຂໍ້ມູນ SQL, ພາສາໂປຣແກຣມ Python, ພາສາ R
  • ທັກສະການແກ້ໄຂບັນຫາ, ສາມາດວິເຄາະຂໍ້ມູນ ແລະ ຫາແນວທາງແກ້ໄຂບັນຫາໄດ້ຢ່າງເໝາະສົມ
