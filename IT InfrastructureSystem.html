Cloud Computing Specialist/Analyst

ແມ່ນຜູ້ຊ່ຽວຊານທີ່ມີໜ້າທີ່ໃນການອອກແບບ ຈັດການ ແລະບຳລຸງຮັກສາລະບົບ Cloud Computing ເຊິ່ງແມ່ນເຕັກໂນໂລຊີທີ່ໃຫ້ບໍລິການຊັບພະຍາກອນຄອມພິວເຕີ.

    ບົດບາດໃນອົງກອນ:

    ການມີຜູ້ຊ່ຽວຊານດ້ານ Cloud Computing ໃນອົງກອນ ແມ່ນເປັນສິ່ງສຳຄັນທີ່ຊ່ວຍໃຫ້ອົງກອນສາມາດໃຊ້ປະໂຫຍດຈາກລະບົບ Cloud ໄດ້ຢ່າງເຕັມປະສິດທິພາບ ພ້ອມທັງນຳພາທຸລະກິດເຂົ້າສູ່ການແຂ່ງຂັນ ແລະການເຕີບໂຕໃນຍຸກດິຈິຕອນ.

    ທັກສະທີ່ຈຳເປັນ:

  • ທັກສະດ້ານພາສາໂປຣແກຣມ
  • ທັກສະໃນການຈັດການຂໍ້ມູນ Database
  • ທັກສະດ້ານ AI ແລະ Machine Learning
  • ຄວາມເຂົ້າໃຈ ແລະປະສົບການກ່ຽວກັບເຕັກໂນໂລຊີ Cloud Computing
  • ທັກສະໃນການວິເຄາະ ແລະມັກຮຽນຮູ້ສິ່ງໃໝ່ໆ

Developer Engineer

ແມ່ນຜູ້ຊ່ຽວຊານດ້ານການພັດທະນາຊອບແວ ທີ່ເນັ້ນການເຮັດວຽກຮ່ວມກັນລະຫວ່າງທີມພັດທະນາ (Development) ແລະ ທີມປະຕິບັດການ (Operations) ໂດຍເນັ້ນໃຫ້ຂະບວນການພັດທະນາ, ການທົດສອບ ແລະ ການສົ່ງມອບຊອບແວ ເປັນໄປດ້ວຍດີ ແລະ ວ່ອງໄວ ຜ່ານການປະສານງານທີ່ດີ.

    ບົດບາດໃນອົງກອນ:

    ມີບົດບາດສຳຄັນໃນການປະສານງານຮ່ວມກັນລະຫວ່າງທີມພັດທະນາ (Development) ແລະ ທີມປະຕິບັດການ (Operations) ເພື່ອໃຫ້ຂະບວນການພັດທະນາຊອບແວ, ການທົດສອບ, ການປ່ອຍ (Deployment), ແລະ ການດູແລຮັກສາລະບົບ ເຮັດວຽກໄດ້ຢ່າງສະດວກ, ວ່ອງໄວ ແລະ ປອດໄພຍິ່ງຂຶ້ນ.

    ທັກສະທີ່ຈຳເປັນ:

  • ທັກສະການຂຽນໂປຣແກຣມ ແລະ ສະຄຣິບ( Script )
  • ຄວາມຮູ້ດ້ານລະບົບ ແລະ ໂຄງສ້າງພື້ນຖານ
  • ທັກສະການໃຊ້ເຄື່ອງມື CI/CD
  • ຄວາມຮູ້ດ້ານການຈັດການ Container ແລະ ການປັບຂະໜາດ
  • ທັກສະການກວດກາ ແລະ ການບຳລຸງຮັກສາ
  • ທັກສະການຮັກສາຄວາມປອດໄພຂອງລະບົບ

⁠Information Security Analyst

ນັກວິເຄາະຄວາມປອດໄພທາງໄຊເບີທີ່ເປັນຜູ້ຊ່ຽວຊານດ້ານຄວາມປອດໄພຂອງລະບົບຄອມພິວເຕີ, ເຄືອຂ່າຍ ແລະ ຂໍ້ມູນສຳຄັນຂອງອົງກອນ.

    ບົດບາດໃນອົງກອນ:

    ແມ່ນຕົວຊ່ວຍທີ່ເສີມສ້າງຄວາມປອດໄພທາງໄຊເບີໃຫ້ແກ່ອົງກອນ ແລະ ກ່ຽວຂ້ອງກັບຄວາມໝັ້ນຄົງຂອງອົງກອນນັ້ນໆອີກດ້ວຍ

    ທັກສະທີ່ຈຳເປັນ:

  • ຄວາມຮູ້ດ້ານຄວາມປອດໄພທາງໄຊເບີ (CyberSecurity)
  • ທັກສະການວິເຄາະ ແລະ ການແກ້ໄຂບັນຫາ
  • ທັກສະການໃຊ້ເຄື່ອງມືຄວາມປອດໄພ (Security Tools Proficiency)
  • ຄວາມເຂົ້າໃຈໃນລະບົບ ແລະ ເຄືອຂ່າຍ
