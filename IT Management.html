IT Project Manager

ພວກເຂົາມີຫນ້າທີ່ຮັບຜິດຊອບໃນການວາງແຜນ, ປະຕິບັດ, ແລະການຄວບຄຸມໂຄງການເຕັກໂນໂລຊີຂໍ້ມູນຂ່າວສານ (IT) ເພື່ອຮັບປະກັນວ່າພວກເຂົາສໍາເລັດຕາມເວລາ, ງົບປະມານ, ແລະສອດຄ່ອງກັບເປົ້າຫມາຍທີ່ກໍານົດໄວ້. ພວກເຂົາເຮັດວຽກເປັນທີມງານ.

    ບົດບາດໃນອົງກອນ:

    ການວາງແຜນດຳເນີນແລະການຄວບຄຸມໂຄງການເຕັກໂນໂລຊີຂໍ້ມູນຂ່າວສານໃຫ້ສຳເລັດຕາມເປົ້າໝາຍທີ່ກຳນົດໄວ້

    ທັກສະທີ່ຈຳເປັນ:

  • ຕິດຕໍ່ສື່ສານກັບທີມງານ, ພາກສ່ວນກ່ຽວຂ້ອງ
  • ຄວາມສາດມາດໃນການວິເຄາະປັນຫາ ຫາແນວທາງແກ້ໄຂແລະຕັດສິນໃຈ
  • ຄວາມເຂົ້າໃຈພື້ນຖານຂອງເຕັກໂນໂລຊີຂໍ້ມູນຂ່າວສານ, ການພັດທະນາຊອບແວ, ແລະການຄຸ້ມຄອງລະບົບ

⁠⁠IT Manager

ມີໜ້າທີ່ດູແລແລະຈັດການໂຄງສ້າງພື້ນຖານດ້ານເຕັກໂນໂລຊີຂໍ້ມູນຂ່າວສານ ຂອງອົງກອນເພື່ອໃຫ້ການດຳເນີນງານເປັນໄປຢ່າງມີປະສິດທິພາບແລະສອດຄ້ອງກັບເປົ້າໝາຍດ້ານທຸລະກິດ.

    ບົດບາດໃນອົງກອນ:

    ການດູແລແລະຈັດການດ້ານໄອທີລວມເຖິງການວາງແຜນ ຈັດການແລະປະເມີນຂະບວນການເຮັດວຽກໄອທີແລະຂໍ້ມູນເອເລັກໂຕນິກ

    ທັກສະທີ່ຈຳເປັນ:

  • ການຄຸ້ມຄອງຄວາມສ່ຽງຄວາມສາມາດໃນການປະເມີນແລະການຄຸ້ມຄອງຄວາມສ່ຽງທີ່ອາດມີຕໍ່ລະບົບ IT
  • ຄວາມສາມາດໃນການປັບຕົວກັບການປ່ຽນແປງຂອງເຕັກໂນໂລຊີແລະສະພາບແວດລ້ອມ.
  • ທັກສະການເຈລະຈາຄວາມສາມາດໃນການເຈລະຈາກັບຜູ້ຂາຍແລະຜູ້ໃຫ້ບໍລິການພາຍນອກ.
  • ຄວາມຮູ້ທາງດ້ານກົດໝາຍທາງດ້ານໄອທີຄວາມເຂົ້າໃຈກ່ຽວກັບກົດໝາຍ ແລະ ລະບຽບການທີ່ກ່ຽວຂ້ອງກັບການນຳໃຊ້ ແລະການເກັບຮັກສາຂໍ້ມູນ

Product Manager

ຜູ້ທີ່ມີໜ້າທີໃນການຮັບຜິດຊອບໃນການດູແລຮັກສາຜະລິດຕະພັນຕັ້ງແຕ່ເລີ່ມຕົ້ນຈົນເຖິງການປ່ອຍຜະລິດຕະພັນອອກສູ່ຕະຫຼາດ

    ບົດບາດໃນອົງກອນ:

    ກໍານົດທິດທາງແລະຍຸດທະສາດຂອງຜະລິດຕະພັນ, ຈາກການວາງແຜນແລະການພັດທະນາເພື່ອເປີດຕົວແລະຕິດຕາມ. ພວກເຂົາເຮັດຫນ້າທີ່ເປັນ "CEO" ຂອງຜະລິດຕະພັນ, ເບິ່ງແຍງທຸກໆດ້ານເພື່ອຮັບປະກັນຜະລິດຕະພັນຕອບສະຫນອງຄວາມຕ້ອງການຂອງລູກຄ້າແລະບັນລຸເປົ້າຫມາຍທຸລະກິດ.

    ທັກສະທີ່ຈຳເປັນ:

  • ມີຄວາມຮູ້ແລະຄວາມເຂົ້າໃຈກ່ຽວກັບຫຼັກການການອອກແບບ (Design Thinking),ປະສົບການຂອງຜູ້ໃຊ້(UX/UI), prototyping,ແລະບັນຫາພື້ນທີ່ການແກ້ໄຂ
  • ຕ້ອງສາມາດສື່ສານວິໄສທັດຜະລິດຕະພັນຢ່າງມີປະສິດທິພາບ, ອະທິບາຍຄວາມຕ້ອງການຂອງລູກຄ້າ, ແລະປະສານງານກັບທີມງານຕ່າງໆ.
  • ຕ້ອງ​ສາ​ມາດ​ເປັນ​ຜູ້​ນໍາ, ສ້າງແຮງບັນດານໃຈ ແລະ​ ກະ​ຕຸ້ນ​ທີມ​ງານ​ເພື່ອ​ບັນ​ລຸ​ເປົ້າ​ຫມາຍ​ຂອງ​ຕົນ​.
  • ຕ້ອງມີຄວາມສາມາດໃນການກໍານົດບັນຫາ, ວິເຄາະສາເຫດ, ແລະຊອກຫາວິທີແກ້ໄຂຢ່າງໄວວາແລະມີປະສິດທິພາບ.
